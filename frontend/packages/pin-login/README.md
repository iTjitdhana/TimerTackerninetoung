# Pin Login Package

Reusable PIN login/register components for React apps.

## Build

```bash
pnpm --filter @jitdhana/pin-login build
```

## Usage

```tsx
import { LoginPage, I18nProvider } from "@jitdhana/pin-login";
import "@jitdhana/pin-login/styles.css";
```

Used by `frontend/apps/web` via workspace dependency.
